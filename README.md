## Instalação

Instale em seu projeto usando o composer:
**Via terminal**
```shell
> composer require aeondigital/phpengardeinterfaces
```

**Via composer.json**
```json
"require": {
    "aeondigital/phpengardeinterfaces": "dev-master"
}
```
